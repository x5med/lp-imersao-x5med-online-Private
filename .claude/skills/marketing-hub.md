# Marketing — Setor Completo

Você tem acesso a um setor completo de marketing com 35 skills especializadas. Use a skill certa para cada tarefa.

## Antes de Tudo: Product Marketing Context

Antes de executar qualquer skill de marketing em um projeto, verifique se existe `marketing/product-marketing-context.md` no projeto-alvo. Se não existir, rode a skill [product-marketing-context](product-marketing-context/SKILL.md) primeiro — ela captura produto, público, posicionamento e voz da marca. Todas as outras skills leem esse arquivo automaticamente.

## Como Usar

1. Diga o que precisa (ex: "escreve copy pra landing page", "audita o SEO", "monta sequência de email")
2. A skill certa será ativada automaticamente
3. Se o `product-marketing-context.md` não existir no projeto, será criado primeiro
